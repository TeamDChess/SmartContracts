# DChessSmartContracts

DChess is the first NFT-Based Play-to-Earn Chess Game ever created. Its economy is governed by two BEP20 Tokens (QUEEN & KING) and one BEP721 Token (BOARD). These contracts were written using openzeppelin@3.4.0 version and compiled using solidity@0.7.0. Its deployment and migration was using Truffle.
